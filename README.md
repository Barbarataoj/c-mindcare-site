# MindCare Site

A modern website built with Astro for MindCare mental health platform.

## Project Structure

```
├── public/
│   └── favicon.svg
├── src/
│   ├── layouts/
│   │   └── BaseLayout.astro
│   └── pages/
│       ├── index.astro
│       ├── about.astro
│       ├── platform.astro
│       ├── safety.astro
│       └── contact.astro
├── astro.config.mjs
├── package.json
└── README.md
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Pages

- **Home** (`/`) - Landing page
- **About** (`/about`) - About MindCare
- **Platform** (`/platform`) - Platform features
- **Safety** (`/safety`) - Safety information
- **Contact** (`/contact`) - Contact form
